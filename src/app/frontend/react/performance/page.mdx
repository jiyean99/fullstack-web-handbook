# 성능 최적화 (Performance)

## 1. 왜 필요한가

불필요한 리렌더링은 UI 버벅거림을 유발합니다.

## 2. 핵심 개념 요약

- `React.memo` — 동일 props에서 리렌더 방지
- `useMemo` — 비싼 계산 메모이제이션
- `useCallback` — 함수 참조 안정화
- `Suspense` + `lazy` — 코드 스플리팅

## 3. 코드 예시

```ts
import { memo, useMemo } from 'react'

const ExpensiveList = memo(({ items }: { items: string[] }) => {
  const sorted = useMemo(() => [...items].sort(), [items])
  return <ul>{sorted.map(i => <li key={i}>{i}</li>)}</ul>
})
```

## 4. 실무 팁

- 최적화는 프로파일링 후 적용 (섣부른 최적화 금지)
- React DevTools Profiler로 실제 병목 확인
