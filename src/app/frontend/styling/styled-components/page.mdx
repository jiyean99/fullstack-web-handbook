# styled-components

## 1. 왜 필요한가

CSS-in-JS로 컴포넌트 단위 스타일 캡슐화와 동적 스타일링이 가능합니다.

## 2. 핵심 개념 요약

- **Tagged Template Literal** 기반 CSS 작성
- **ThemeProvider** — 전역 디자인 토큰 공유
- **Props 기반 동적 스타일** — 조건부 CSS
- **Server Component 주의** — App Router에서는 Client Component에서만 사용

## 3. 코드 예시

```ts
'use client'
import styled, { css } from 'styled-components'

const Button = styled.button<{ $primary?: boolean }>`
  padding: 0.5rem 1rem;
  border-radius: 0.375rem;
  font-weight: 500;
  
  ${({ $primary }) => $primary ? css`
    background: #2563eb;
    color: white;
  ` : css`
    background: white;
    border: 1px solid #d1d5db;
  `}
`
```

## 4. 실무 팁

- transient props(`$prefix`)로 DOM에 전달되지 않아야 할 props 구분
- styled-components 6.x에서 `attrs()` API 변경 주의
