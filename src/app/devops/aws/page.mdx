# AWS

## 1. 왜 필요한가

클라우드 인프라의 핵심 서비스를 이해하면 배포 자동화가 쉬워집니다.

## 2. 핵심 개념 요약

- **EC2**: 가상 서버
- **S3**: 오브젝트 스토리지 (정적 파일, 이미지)
- **RDS**: 관리형 관계형 DB
- **ECS / EKS**: 컨테이너 오케스트레이션
- **CloudFront**: CDN

## 3. 실무 팁

- 프리 티어로 실습 가능 (EC2 t2.micro, RDS db.t3.micro)
- IAM 역할 최소 권한 원칙 적용
- 보안 그룹으로 인바운드/아웃바운드 트래픽 제어
